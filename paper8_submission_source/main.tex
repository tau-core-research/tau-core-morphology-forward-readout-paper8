\begin{abstract}
Paper 1 found that residual-blind external structural disturbance labels are associated with increased low-acceleration residual scatter in SPARC galaxies. Paper 2 reversed the question and showed that fixed residual-shape features can infer those disturbance labels better than shuffled controls, while remaining non-unique with respect to MOND-simple and empirical RAR-like baselines. Paper 3 then built a candidate/control framework and introduced an endpoint-conditioned required-$S_\tau$ diagnostic: what preparation/readout factor would be needed to absorb a measured residual? This Paper 8 proposal turns that inverse diagnostic into a forward validation gate. The proposed gate predeclares a residual-blind morphology class $K_g$, maps it to a Tau Core morphology-conditioned readout shell, computes a forward $\delta g^K$ without using required $S_\tau$ as a predictor, and compares matched-family residual structure against wrong-family, shuffled-label, Newtonian baryonic, MOND-simple, and empirical RAR controls. The permitted claim is methodological: the morphology-matched forward-readout gate is a falsifiable, Tau-Core-specific next endpoint for the Paper 1--3 program. This paper does not claim that Tau Core has been empirically validated, that a universal galaxy law has been derived, or that the matched-family endpoint has already beaten MOND/RAR on real SPARC data.
\end{abstract}

\section{Purpose and claim boundary}

The first three SPARC residual papers define a cautious progression. Paper 1 asks whether external structural disturbance is associated with residual scatter \cite{Paper1Package}. Paper 2 asks whether residual-shape features can recover external disturbance labels better than shuffled labels \cite{Paper2Package}. Paper 3 turns the association into a candidate/control framework and asks what $S_\tau$ would be required to absorb the observed residual \cite{Paper3Package}.

That Paper 3 $S_\tau$ diagnostic is useful but partly inverse. It asks:
\[
\text{given the measured endpoint, what } S_\tau(R) \text{ would absorb it?}
\]
The present paper defines the next gate:
\[
\begin{aligned}
&\text{residual-blind morphology class }K_g\\
&\quad\longrightarrow\text{ forward readout }\delta g^K
\text{ before endpoint scoring.}
\end{aligned}
\]

The main claim is deliberately narrow:
\begin{quote}
The Tau Core gravity bridge can sharpen Papers 1--3 by replacing inverse required-$S_\tau$ residual absorption with a morphology-matched forward-readout candidate/control gate.
\end{quote}

This paper does not claim:
\begin{itemize}[leftmargin=*]
\item that Tau Core is proven;
\item that the morphology-matched gate has already won on real SPARC endpoints;
\item that MOND, RAR, or Newtonian baryonic baselines have been superseded;
\item that non-axisymmetric morphology classes can be fully tested from one-dimensional rotation curves alone;
\item that the morphology families below are the final physical taxonomy.
\end{itemize}

\section{From Paper 3 inverse diagnostics to forward readout}

Paper 3 used an endpoint-conditioned diagnostic of the form
\[
S_{\tau,\mathrm{req}}(R)=
\frac{V_{\rm obs}(R)/V_N(R)-1}{\alpha\ln(1+a_0/a_N(R))}.
\]
This quantity is descriptive. Because it uses $V_{\rm obs}$, it cannot be treated as an independent predictor. Its correct role is to identify what kind of future nonleaky rule would be required.

The forward replacement is:
\[
K_g \longrightarrow \mathcal{F}_{K_g}
\longrightarrow \delta g^{K_g}(R)
\longrightarrow \mathrm{Score}(g_{\rm Newt}+\delta g^{K_g}).
\]
Here $K_g$ is a morphology label assigned before endpoint scoring, $\mathcal{F}_{K_g}$ is the predeclared readout family, and $\delta g^{K_g}$ is computed without using $S_{\tau,\mathrm{req}}$, endpoint residual gain, or post-hoc family selection.

\section{MORPHOLOGY-MATCHED-FORWARD-READOUT-GATE}

For a galaxy $g$, let
\[
K_g\in\mathcal{K}
\]
be a residual-blind morphology label. Let $\mathcal{F}_K$ denote the Tau Core readout shell assigned to morphology class $K$. Let
\[
R_{g,K}
\]
be the residual score after applying the forward readout family $\mathcal{F}_K$ to galaxy $g$ under a frozen geometry and amplitude policy.

The matched-family score is
\[
R_{g,\mathrm{matched}}:=R_{g,K_g}.
\]
The wrong-family score is
\[
R_{g,\mathrm{wrong}}:=
\frac{1}{|\mathcal{K}|-1}
\sum_{K\neq K_g}R_{g,K}.
\]
The central paper-level endpoint can be written as
\[
\Delta_{\rm matched}
=
\mathrm{RMS}_{g}\!\left(R_{g,\mathrm{wrong}}\right)
-
\mathrm{RMS}_{g}\!\left(R_{g,\mathrm{matched}}\right).
\]
An equivalent rank endpoint is
\[
\mathrm{rank}_{g}(K_g)
=
\mathrm{rank}\bigl(R_{g,K_g}\;\mathrm{among}\;\{R_{g,K}:K\in\mathcal{K}\}\bigr).
\]

The gate passes only if the predeclared matched family improves residual structure more than:
\begin{enumerate}[leftmargin=*]
\item wrong morphology families;
\item shuffled morphology labels;
\item Newtonian baryonic baseline;
\item MOND-simple baseline;
\item empirical RAR-like baseline.
\end{enumerate}

This is Tau-Core-specific in a way that ordinary MOND/RAR comparisons are not: the discriminating claim is not merely lower residuals, but morphology-indexed readout specificity.

The positive paper-level endpoint is not:
\[
\text{Tau Core fits every galaxy better.}
\]
The first meaningful endpoint is:
\[
\text{the correct predeclared morphology family ranks better than wrong families.}
\]
This is why shuffled-$K$ nulls are mandatory. Without the shuffled-label null, a morphology-indexed family could accidentally behave like a generic flexible residual smoother.

\section{Available-data Tau-proxy preflight}

The repository also includes a first available-data preflight of the matched-family idea. This is not the final Paper 8 endpoint, because the family functions are still proxy functions built from the locally available Tau-proxy channel rather than the final morphology-specific $\delta g^K$ readout shells. The preflight nevertheless tests the important next question: whether a metadata-proxy morphology family assigned before scoring performs better than wrong Tau-proxy families on held-out galaxies.

In the current 175-galaxy point-level packet, family amplitudes are fit on the train split only. On the 44-galaxy holdout split, the morphology-matched proxy beats the wrong-family mean in 0.568 of galaxies, beats the TPG/v6 comparator in 0.636 of galaxies, and beats the MOND comparator in 0.614 of galaxies. These numbers are recorded as a preparation result, not as empirical validation of Tau Core. They motivate replacing the generic Tau proxy with source-native morphology-family readout shells and a shuffled-label null in the final endpoint.

\section{Minimal numerical protocol}

The minimal public endpoint protocol is:
\begin{enumerate}[leftmargin=*]
\item Assign an external or residual-blind morphology label $K_g$ to each galaxy.
\item Select a formula shell $\mathcal{F}_{K_g}$ from the predeclared registry.
\item Fix geometry parameters from morphology, not from residual endpoints.
\item Use either a frozen amplitude rule or a tightly bounded amplitude fit.
\item Compute the matched-family score.
\item Compute wrong-family scores for all other candidate families.
\item Run shuffled-$K$ nulls.
\item Compare against Newtonian baryonic, MOND-simple, and empirical RAR baselines.
\end{enumerate}

For SPARC-only first-pass work, the cleanest initial family set is:
\[
\{K_{\rm tail},K_{\rm exp},K_{\rm compact},K_{\rm ring},K_{\rm flare}\}.
\]
The $m=1$ and $m=2$ branches should be treated as caveated proxy rows in one-dimensional rotation curves and as real tests only when velocity-field products are available.

\section{Why this would be a stronger Tau Core test}

The new gate is stronger than a residual association because it creates a family-selection burden. A theory with morphology-indexed readout families should not only reduce residuals; it should reduce them more when the correct morphology family is used than when a wrong family is used. MOND-simple and empirical RAR baselines are strong comparators, but they do not naturally provide a predeclared morphology-family rank test of this kind.

The result that would matter is therefore:
\[
\Delta_{\rm matched}>0
\]
with a rank distribution in which $K_g$ is systematically near the top, under residual-blind morphology labels and with shuffled-label controls. If this fails, the bridge loses specificity. If it passes, the Paper 1--3 residual-disturbance program becomes a forward-predictive morphology-readout program.

\section{Conclusion}

Paper 3 is the natural launching point for this forward gate because it already contains the candidate/control framework and the required-$S_\tau$ diagnostic. The present contribution is to state the next gate in a falsifiable form:
\[
\text{morphology label }K_g
\to
\text{formula shell }\mathcal{F}_{K_g}
\to
\delta g^{K_g}
\to
\text{matched vs wrong family endpoint}.
\]

The claim-safe breakthrough target is not a universal fit claim. It is a morphology-specific prediction claim: the correct Tau Core morphology readout family, chosen before endpoint scoring, should improve residual structure more than wrong morphology families and shuffled morphology labels, while remaining competitive against Newtonian baryonic, MOND-simple, and empirical RAR baselines.

That is the first clean endpoint where the Tau Core bridge can become more than residual commentary.
